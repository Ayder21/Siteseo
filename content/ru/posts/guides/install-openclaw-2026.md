# Почему OpenClaw, а не AutoGPT?

В 2026 году рынок AI-агентов изменился. AutoGPT остался игрушкой для гиков, а **OpenClaw** стал стандартом для автоматизации работы.

Почему?
1. **Безопасность.** Он работает локально. Ваши ключи API и данные не улетают на чужие сервера.
2. **Модульность.** Хотите подключить Telegram? Одну команду. Хотите управление умным домом? Еще одну.
3. **Open Source.** Полностью бесплатный и открытый код.

## Системные требования

* **Node.js:** Версия 20 или выше.
* **RAM:** Минимум 4GB (8GB рекомендуется).
* **OS:** Windows 10/11, macOS (M1/M2/M3 летает), Linux (Ubuntu 24.04+).

## Шаг 1: Установка Node.js

Если у вас нет Node.js, скачайте его с [официального сайта](https://nodejs.org/). Рекомендуем версию **LTS**.

Проверьте установку в терминале:
```bash
node -v
npm -v
```

## Шаг 2: Установка OpenClaw

Откройте терминал (PowerShell на Windows, Terminal на Mac) и введите:

```bash
npm install -g openclaw
```

## Шаг 3: Первый запуск

Запустите агент командой:

```bash
openclaw
```

Вас встретит мастер настройки. Он спросит:
1. Какой язык использовать? (Выбирайте Русский).
2. Какие API ключи есть? (OpenAI, Anthropic, или локальные модели через Ollama).
